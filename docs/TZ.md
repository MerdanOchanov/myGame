# Техническое Задание (ТЗ) для игры «Survival Turkmenistan»

## Общая идея
Игра использует реальные бесплатные карты Туркменистана. Случайная генерация биомов с флорой и фауной. Игроки видят свою геопозицию, собирают ресурсы, сражаются, крафтят лекарства из ингредиентов с 50+ случайными характеристиками.

## Ключевые механики
- Регистрация: игрок занимает клетку 100м как дом.
- Сбор ресурсов и бой в биомах.
- Крафт лекарств (200+ эффектов, скрытые характеристики, синергия).
- Случайные мировые события.
- Смерть = респавн через 24 часа реального времени.

## Архитектура
- core/: Data, State, ECS, Biome, Crafting
- render/: Phaser scenes, Leaflet map, UI
- systems/: Game systems
