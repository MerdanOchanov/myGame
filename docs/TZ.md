# Техническое Задание (ТЗ) - myGame

## Общая идея
Игра на основе реальной карты Туркменистана. Игроки занимают клетки 100м, собирают ресурсы в биомах, крафтят лекарства из ингредиентов с скрытыми характеристиками.

## Ключевые механики
- Генерация биомов со случайной флорой/фауной
- Геопозиция игрока
- Claim дома (100м клетка)
- Сбор ресурсов, бой с животными
- Crafting системы с 200+ эффектами и синергиями
- Случайные события
- Пермедеат (respawn через день)

## Архитектура
- core/: Data + State + ECS
- render/: Phaser + Leaflet
- systems/: Логика
